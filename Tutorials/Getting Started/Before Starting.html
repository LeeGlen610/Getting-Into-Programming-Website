Before Starting

◄ Backward Foward ►

Before you start programming you'll need to install a few things before installing your IDE:

With Java you'll need to install a JDK & JRE before installing your IDE.

JDK & JRE

  1. Go to these links: JDK and the JRE download the JDK and JRE exe files for your operating system. This will be mainly focusing on the windows version.
  2. Click on the JDK exe file and click on next to everything and let it install the JDK and then click close. That is you installed the JDK. This is a similar process with the JRE.

That is you installed what you need before you install the IDE.

For C++ you can either download Visual Studio which we will get to with the IDE's or use MinGW.

MinGW

  1. Go to this link: MinGW and download the mingw-get-setup.exe and run it.
  2. Click on the exe and click install and let it install to your C: drive. Once it has installed the download manager click on the boxes to these packages.
    • mingw32-base-bin
    • mingw-developer-toolkit-bin
    • mingw32-gcc-g++-bin
    • msys-base-bin

That is you installed what you need before installing the IDE.

To use python on a IDE you'll need to install python.

Python

  1. Go to this link: Python and click download python exe.
  2. Click on the exe and install to the default location. At the end of installation click close.

That is you installed what you need before installing the IDE.
