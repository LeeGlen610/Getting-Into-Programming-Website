Setting Up An IDE

◄ Backward Foward ►

An IDE is where the environment you'll mainly be developing your programs in. Multiple languages will use different IDEs' to be developed on.

There is two IDEs' I would recommend - InteliJ & Eclipse.

InteliJ

  1. Go to this website: InteliJ and download the community edition.
  2. Click on the exe and let it install to the default location and tick .java for associations and 64-bit launcher when asked and then install
  3. Once you've installed the IDE start it and go through the first time start up and pick your UI and then default everything.

That is you ready to use InteliJ.

Eclipse

  1. Go to this website: Eclipse and click on the download button.
  2. Click on the exe and click on Eclipse for Java Developers and then click install. There will be a prompt that will pop-up, just click select all and then accept selected.
  3. Once you've installed the IDE start it and choose where you want to create your workspace to have a place to save all your programs and click launch.

That is you ready to use Eclipse.

Creating A Project

From here on out I'm assuming that you are using InteliJ.

Click on new project and then click Java and then where it says project SDK click on ... and go to - C:\Program Files\Java and click on the JDK and press next and press next again and then save the project wherever you wish to save it and click finish.

Open up the folder with the project name and then click on src, right-click, new and then click class. Name the class and then press enter.

Once you've done coding click on the play button either next to the method or near the top of the IDE.

For C++ we will install Visual Studio.

Visual Studio

  1. Go to this link: Visual Studio and click and download the community edition.
  2. Click on the exe and then press continue and then once it has downloaded what it needed you'll need to choose these three workloads:
    • .NET desktop development
    • Desktop Development with C++
    • Universal Windows Platform developer
    and then press install.
  3. Once it's installed start it up and choose the UI and click Start Visual Studio.

Creating A Project

Click on Create A New Project and then choose empty project and then name the project and specify the location of where the project will be saved to.

Click on the Source Files folder and then right-click and go to add and then new item. Choose C++ file, name it and click add. Then double click on the file.

Once you have coded what you needed to do click on the green plus

For Python we will be using PyCharm.

PyCharm

  1. Go to this website: PyCharm and download the community edition.
  2. Click on the exe and let it install to the default location and tick .py for associations and click 64-bit launcher when asked and then install
  3. Once you've installed the IDE start it and go through the first time start up and pick your UI and then default everything.
  4. Click on configuration and then settings. Then click on Project Interpreter and then click on the gear button and then add. It should pop up with it automatically showing python 3.8 and then press ok. Once it is done installing and updating the python press apply then ok.

Creating A New Project

Click on new project and then click pure python and choose where you're saving your project and click next

Open up the folder with the project name, right-click, new and then click python file. Name the file and then press enter.

Once you've done coding click on the file and then right-click and then click run.
