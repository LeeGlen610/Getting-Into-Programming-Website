Classes

◄ Backward Foward ►

Classes are blueprints for objects. What this means is that every object will have all the attributes and methods declared in a class which then can be used.

There is four classes in the diagram - Animal, Bird, Dog & Cat. We will take these classes in the diagram and implement them into code.

Animal Class

public class Animal{


String name;

int age;

double height;

String type;


String getName(){

return name;

}


void setName(String name){

this.name = name;

}


int getAge(){

return age;

}


void setAge(int age){

this.age = age;

}


double getHeight(){

return height;

}


void setHeight(double height){

this.height = height;

}


String getType(){

return type;

}


void setType(String type){

this.type = type;

}


void speak(){

System.out.println("Hello!");

}

}

Bird Class

public class Bird{


boolean canFly;


boolean canFly(){

return canFly;

}


void setCanFly(){

canFly = !canFly;

}


void speak(){

System.out.println("Tweet!");

}

Dog Class

public class Dog{


void sniff(){

System.out.println("Sniff Sniff!");

}


void speak(){

System.out.println("Bark!");

}

}

Cat Class

public class Cat{


void claws(){

System.out.println("Hiss! The cat has now scratched you!");

}


void speak(){

System.out.println("Meow!");

}

}
