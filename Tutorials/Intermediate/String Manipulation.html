String Manipulation

◄ Backward Foward ►

String is an array of characters that allows you to store text. However string isn't a data type as it is an object. We'll come to objects when we talk about Object-oriented programming. We can manipulate string in a lot of ways.

We can:

  • Find the length of a string.
  • Add two strings together.
  • Change all the characters into uppercase characters.
  • Check to see if two text are equal to each other. - This is different than using the double equals operator as it will check the object instead of the text.

and many more...

public class main{

public static void main(String[] args){

String sentence = "abcdefgh";

System.out.println(sentence.length());

sentence = sentence + "ijklmnopqrstuvwxyz";

System.out.println(sentence);

System.out.println(sentence.toUpperCase());

if (sentence.equals("abcdefjhijklmnopqrstuvwxyz"){

System.out.println("The Alphabet is: " + sentence)

}

}

}
