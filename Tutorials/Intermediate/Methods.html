Methods

◄ Backward Foward ►

Methods is used to be called to run specific code defined by the programmer and that they can allow data to be passed through into the method. It can also pass back data to the original call of the method.

To create a method you'll need to specify the type of data being returned back or specify void if nothing will be returned back, then the name of the method and finally two curly brackets either empty if no data is being passed through or the data type and name of variable being passed through.

public class Main{

public static void main(String[] args){

int number = 10;

Main main = new Main(); //Used to call the method in the Main class

boolean numberBiggerThanNine = validateNumber(number);

System.out.println("Number is bigger than nine.");

}

}

if(number > 9){

return true;
