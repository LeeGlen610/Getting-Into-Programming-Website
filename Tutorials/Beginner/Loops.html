Loops

◄ Backward Foward ►

What a loop will do is for either a set period of time or when a condition is met will repeat specified code over and over again


There are three types of loops for now :

  • Fixed/For loop
  • While loop
  • Do While loop

public class Main(){

int number = 0;


for (int i = 0; i < 10; i++){

System.out.println(number);

number++; // double plus means increment the value inside the variable.

}


while(number < 20){

System.out.println(number);

number++;

}


do{

number = 1;

}while (number != 1); // != means not equal.

}
