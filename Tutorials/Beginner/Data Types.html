Data Types

◄ Backward Foward ►

Primitive data types is used to specify what data you will be using or storing and there is 8 type of primitive data.

Type Description
int Used to store whole numbers - limit is 2,147,483,648 and -2,147,483,648.
long Used to store very large whole numbers. More than likely to use when number can't be stored in an int.
char Used to store one character.
byte Similar to int but can only store from -128 to 127.
short Similar to int but can only store from -32,768 to 32,767.
double Used to store floating point numbers.
float Similarly to double, can be used to store floating point numbers and could be used instead of double if you want to save memory.
boolean Used to store true or false - used like a flag.

Variables are used in junction with data types to store data.

Variables

Variables are used to store data of some type to be used at a later date.

To create a variable you start with the data type you'd want to use and then the variable name usually in camelcase - this mean if the name of the variable is two words for example the number it should be theNumber - then depending or not you are entering data into the variable you'll have an equals and then the data you're entering or close the line of code.

public class Main { public static void main(String[] args){ int number; double doubleNumber; char character; number = 0; System.out.println(number); } }
