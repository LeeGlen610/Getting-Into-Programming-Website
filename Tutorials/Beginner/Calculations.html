Calculations

◄ Backward Foward ►

For calculations you have your addition, subtraction, multiply, divide and modulus.

Addition - +

Subtraction - -

Multiply - *

Divide - /

Modulus - %

public class Main{

public static void main(String[] args){

int number = 5;


number = number + 5; //OR you can do number += 5

System.out.println(number);

number = number - 9; //OR you can do number -= 9

System.out.println(number);

number = number * 4;

System.out.println(number);

number = number / 2;

System.out.println(number);

number = number % 2;

System.out.println(number);

}

}
